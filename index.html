Project Overview

Designed a comprehensive security alert system using micro:bit and various sensors to create an effective home security solution.

Key Components

  • Motion Sensor (PIR): Detects movement in a monitored space
  • Reed Switch: Detects door opening and closing
  • Buzzer: Activates a loud alarm when a threat is detected
  • Telegram Integration: Sends real-time alerts to the owner's phone

Technologies Used

micro:bit Python IoT Sensors Telegram API
